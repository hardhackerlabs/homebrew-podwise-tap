# homebrew-tap

Homebrew tap for [hardhackerlabs](https://github.com/hardhackerlabs) tools.

## Install

```bash
brew tap hardhackerlabs/podwise-tap
brew install podwise
```

## Tools

| Formula | Description |
|---------|-------------|
| [podwise](https://github.com/hardhackerlabs/podwise-cli) | CLI client for [podwise.ai](https://podwise.ai) — turn any podcast episode into AI-powered insights |

## Update

```bash
brew update
brew upgrade podwise
```
